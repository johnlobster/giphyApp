Giphy reading App

Information

This is a gif viewer that fetches gifs on a topic from giphy.com

each topic has a button, click on the button to see gifs on that topic

A still image for each gif is shown, to start the gif moving, click on the image

New topics can be added using the text box marked 'Type to create new button'. This creates the new button, which needs to be clicked to see gifs on that topic

Click the favorites button underneath each image to add the image into the favorites section. Favorite images persist from session to session. Click on the remove button if you don't want that image in favorites any more
